# Atividade-Lembrete

Lógica Principal: configurar readline, criar variáveis globais
(vetor de lembretes), e executar.
Módulos:

• Executar: o menu com as opções

• Adicionar: perguntar o lembrete e prazo e adicionar um objeto

com essas informações (lembrete, prazo, concluído) no vetor.

• Listar: verificar se tem algum lembrete cadastrado, se tiver, listar

usando forEach.

• Editar: listar, selecionar um lembrete, perguntar as novas

informações do lembrete, e salvar na posição.

• Marcar como Concluido: listar, selecionar um lembrete e marcar

concluído como true.
